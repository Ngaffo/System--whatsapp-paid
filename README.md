# 💰 WhatsApp Paid Access System

A simple web application that allows users to pay before accessing a private WhatsApp group.

---

## 🚀 Features

- User information form (name and phone number)
- Payment instructions (MTN MoMo)
- Countdown timer before payment confirmation
- WhatsApp redirection with pre-filled message
- Controlled access to WhatsApp group

---

## 🧠 How it works

1. User enters their name and phone number
2. Clicks on **"Pay Now"**
3. Payment instructions are displayed
4. User completes payment via Mobile Money
5. Waits for countdown to finish
6. Clicks **"Payment Completed"**
7. Redirected to WhatsApp with a pre-filled message
8. Admin verifies payment and sends group link

---

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript

---


## ⚠️ Important Note

This system uses manual payment verification to prevent unauthorized access. The WhatsApp group link is only shared after payment confirmation.
